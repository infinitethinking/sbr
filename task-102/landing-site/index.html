FAQs

Is coaching support included?

No. The Seasonal Playbook is a self-serve operating system built to replace 1:1 coaching for self-directed athletes.

How do I access the files?

Immediate download (CSV, Markdown, HTML) plus GitHub access for updates. Works offline in your own trackers.

What if I’m racing multiple distances?

All templates share the same backbone—switch lanes (Finish/Fast/Podium) and distance tab without rebuilding from scratch.
